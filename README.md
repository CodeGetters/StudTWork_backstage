## Install

require:

- node >= `16`

```shell
$ git clone https://github.com/CodeGetters/studTWork.git

# Recommend to use yarn
$ yarn install

# or using npm

$ npm install

# There may be some problems with using pnpm â—
$ pnpm install

```

## Usage

```shell
# dev
$ yarn dev

# build
$ yarn build

# Or maybe you want to package a background project with mock data
$ yarn build:mock
```
